# 🌌 Steam Custom Game Profile
A high-quality, aesthetic custom skin for your Steam Library. This project features a monochrome manga style combined with a deep space atmosphere, centered around the theme of "Perfection" (完璧).

## 📥 How to Download
1. Go to the [Releases] section on the right side of this page.
2. Find the latest version and look for the Assets dropdown menu.
3. Download the .zip file (e.g., Custom_Game_Profile_Steam.zip).
4. Extract the archive to any folder on your computer.

## 🛠️ How to Install (Steam Setup)
**To apply these assets to a specific game in your library, follow these steps:**
1. Set Vertical Cover (Library Grid)
2. Right-click the game in your Library list or on its Grid icon.
3. Select Manage > Set custom artwork.
4. Upload the oblozhka.png file.
5. Set Background & Logo (Game Page).
6. Click on the game in your library to open its main page.
7. Right-click anywhere on the current background banner at the top.
8. Select Set Custom Background and upload fon.png (or fon1.png for the geometric version).
9. Right-click the background again and select Set Custom Logo.
10. Upload Logo.png (or Logo1.png).
